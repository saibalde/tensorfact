class TtVector¶

TT representation of a multidimensional vector

A TT-vector is a memory-efficient representation of a multidimensional array \(v(i_0, \ldots, i_{d - 1})\) where each of the entries are computed as

\[ v(i_0, \ldots, i_{d - 1}) = v_0(i_0) \cdots v_{d - 1}(i_{d - 1}) \]
Here \(v_k(i_k)\) is the \(i_k\)-th slice of the 3D array \(v_k\), also referred to as the \(k\)-th TT core of \(v\). Each of these slices are \(r_k \times r_{k + 1}\) dimensional matrices, with \(r_0 = r_d = 1\). Assuming \(n_k \sim n\) and \(r_k \sim r\), this reduces the storage complexity \(\mathcal{O}(n^d)\) of the full tensor to \(\mathcal{O}(d n r^2)\) in the TT format.
